## Smart Fridge
# Features
1) Automatic item scanner + expiry date (barcode + computer vision + local database of EAN barcodes https://wiki.openfoodfacts.org/Reusing_Open_Food_Facts_Data#Python)
2) Live item stock indicator (weight sensors)
3) Automatic food expiry checker (gas sensors + maybe computer vision to identify expiry date on packaging)
